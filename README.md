# About

Sketches made using [p5.js](https://p5js.org), displayed in a [Gatsby JS](https://www.gatsbyjs.org/) site.

## To Run Locally:

- Download this repo
- Run `npm install`
- Run `gatsby develop`
- Open a web browser to (http://localhost:8000)
